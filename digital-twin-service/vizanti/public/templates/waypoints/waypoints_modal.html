Waypoint Mission

Status: Ok.

Defines a sequence of poses and publishes them as a nav_msgs/Path or geometry_msgs/PoseArray.

X/Y Axis: Edit horizontal positions.

  • Add Points: Click on the view area.
  • Delete Points: Tap an existing point.
  • Move Points: Drag an existing point to reposition it.
  • Snap to Grid: Hold the Shift key while dragging to snap to the pixel grid.

Z Axis: Edit vertical positions.

  • Set Height: Click and drag a point to set a rough Z value.
  • Fine Adjust: Release and drag a point a second time to fine adjust its height.

When editing X/Y/Z positions, click the icon again to stop editing.

Clear Path

Tip: If you're using a nav stack that doesn't use Paths or PoseArrays, but will accept simple goals, see the demo node at rosrun vizanti waypoints_to_simple_goals.py
