Button

Status: Ok.

Multifunctional button. Behavior depends on the selected topic/service type:

  • std_msgs/Empty: Sends an Empty message on click.
  • std_srvs/Trigger: Calls a Trigger service.
  • std_srvs/Empty: Calls a Empty service.
  • std_msgs/Bool: Displays topic state and sends a negated value on click.
  • std_srvs/SetBool: Negates internal boolean with each call, displays state.

Click the icon to trigger. Long press to reopen this menu.
